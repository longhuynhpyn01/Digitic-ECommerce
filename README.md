# Install Dependencies

**For Backend** - `npm i`

**For Frontend** - `cd frontend` ` npm i`

## Env Variables

Make Sure to Create a .env file in backend/config directory and add appropriate variables in order to use the app.

**Essential Variables**
PORT
MONGODB_URI
JWT_SECRET
JWT_EXPIRE
COOKIE_EXPIRE
SMPT_HOST
SMPT_PORT
SMPT_SERVICE
SMPT_MAIL
SMPT_PASSWORD
CLOUDINARY_NAME
CLOUDINARY_API_KEY
CLOUDINARY_API_SECRET
